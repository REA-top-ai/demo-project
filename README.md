# Приложение - пример проекта на творческий рейтинг

## Идея приложения

1. Впервые вошедший пользователь заполняет анкету о любимом визуальном стиле и регионе проживания
2. Приложение запрашивает погоду в интересующей пользователя территории
3. Нейросеть рисует пользователю картинку с отображением погоды в любимом стиле
